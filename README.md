2018mcm1
========

A backup of my homework. 2018 美赛本校第1次选拔赛。

Dependent
---------

1.  A LaTex distribution. Such as [texlive].

Install
-------

``` {.zsh}
git clone git@github.com:Freed-Wu/2018mcm1
cd 2018mcm1
latexmk -pvc main.tex
```
